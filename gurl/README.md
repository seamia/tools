# gurl

> it's like `curl` on mesculine =)

## Install

```sh
# install go, dep
go get github.com/seamia/tools
cd ${GOPATH}/src/github.com/seamia/tools/gurl
dep ensure
go build
go install
ln -s ${GOPATH}/src/gurl /usr/local/bin/gurl
```

## Example

```shell script
see ${GOPATH}/src/github.com/seamia/tools/gurl/sample
```
